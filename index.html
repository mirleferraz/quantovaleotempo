Estimativa
Configurar Google Maps
Conecte sua Google Maps API Key para obter dados reais de rota, trânsito em tempo real e autocomplete de endereços. Sem a key, o sistema usa IA como fallback.
APIs necessárias no projeto GCP
Places API — autocomplete de endereços
Directions API — rotas e tempo de trajeto
Distance Matrix API — tráfego em tempo real
Verifique estes pontos
1. Faturamento ativo — O GCP exige cartão cadastrado mesmo no plano gratuito. console.cloud.google.com/billing
2. Places API ativada — APIs e Serviços → Biblioteca → buscar "Places API" → Ativar
3. Directions API ativada — mesma tela, buscar "Directions API" → Ativar
4. Sem restrições de domínio — Credenciais → editar a key → "Restrições de aplicativo" → Nenhuma
5. Aguarde 2–5 minutos após ativar as APIs pela primeira vez
Como obter a key:
1. Acesse console.cloud.google.com
2. Crie um projeto → Ative as 3 APIs acima
3. Gere uma API Key em "Credenciais"
4. Recomende restringir a key ao seu domínio
Claude IA (fallback de rotas)
Necessária para calcular rotas e autocomplete quando o Google Maps não estiver configurado. Obtenha em console.anthropic.com.
